Podcast Dataset¶

By Alhanoof Alnajashi¶

Podcast dataset is chosen since its very trending these days and people love to know more about them and it can't be better than data in visual!

Srource of the data Here This dataset includes the metadata of (almost) all podcast episodes that were published in December 2017.

Information about the data set:

it has to CSV file podcast.csv and episodes.csv

in podcast.csv there are 9 attributes

uuid : is the podcast id in the ListenNotes.com.
title: Podcast title.
image: The image of this podcast.
description: Podcast description, in HTML format.
language: Language of this podcast.
categories: Genres of this podcast.
website: Website of this podcast.
author: The publisher of this podcast.
itunes_id: iTunes id of this podcast.

in episodes.csv there are 7 attributes

title: Episode title.
audio: Audio url.
audio_length: Audio length in seconds.
description: Episode description.
pub_date: Published date of this podcast.
uuid: is the episode id in the ListenNotes.com.
podcast_uuid: uuid of the podcast that contains this episode.
